Module settings

Before using this module, you need to create the Auto-client application in Privat 24 for business and get its Token and ID.

To do this, log in to the application, select the legal entity you are interested in (if you have several).

Next, in the left menu, select "Service Catalog", select Auto-client/Connect/Open.

Click add "Autoclient". Do not change the first and third tabs, and in the second tab select the accounts you are interested in.

Save.

At the bottom of the "Autoclient" menu, find "API for developers", copy the Token and ID.

Next, you should go to Odoo, select the Applications category, click the Refresh Application Lists button (located on the top left) and enter "Private" in the search bar, install the modules you found. The next step is to go to the Accounts/Settings/Logs section.

Create a new journal and enter the necessary data. Adjust debit and credit accounts. Enter the Token and ID.

After entering all the settings, you can receive bank statements automatically according to the specified schedule.
