Тільки користувацькі повідомлення в чаті


Цей модуль покращує читабельність чату Odoo, фільтруючи системні повідомлення та показуючи лише ті, що створені користувачами. Він допомагає підтримувати чисту та зосереджену історію комунікацій, приховуючи автоматичні сповіщення, електронні листи та повідомлення про зміни.

Основні можливості:

  • Фільтрація повідомлень: Показ лише повідомлень, створених реальними користувачами
  • Чистий інтерфейс: Приховування автоматичних системних повідомлень та сповіщень
  • Універсальна сумісність: Працює з усіма моделями Odoo, що використовують чат
  • Нульова конфігурація: Працює одразу після встановлення
  • Покращена читабельність: Фокус на реальному спілкуванні користувачів

Використання

Після встановлення:

  1. Всі повідомлення, створені користувачами, залишаються видимими в чаті
  2. Системні повідомлення автоматично приховуються:
    • Автоматичні сповіщення електронної пошти
    • Записи про відстеження змін
    • Системні сповіщення
    • Заплановані нагадування
  3. Не потрібна додаткова конфігурація - працює одразу після встановлення

Відстеження помилок

Помилки відстежуються на https://kitworks.systems/requests. У разі виникнення проблем, будь ласка, перевірте, чи про вашу проблему вже не повідомлялося.
