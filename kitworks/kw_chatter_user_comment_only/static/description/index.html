Chatter User Comment Only


This module enhances the clarity of Odoo chatter by filtering out system-generated messages and showing only those created by users. It helps maintain a clean and focused communication history by hiding automated notifications, emails, and logging messages.

Key features include:

  • Message Filtering: Show only messages created by actual users
  • Clean Interface: Hide automated system messages and notifications
  • Universal Compatibility: Works with all Odoo models using chatter
  • Zero Configuration: Works immediately after installation
  • Improved Readability: Focus on actual user communication

Usage

After installation:

  1. All user-created messages remain visible in chatter
  2. System-generated messages are automatically hidden:
    • Automated email notifications
    • Change tracking logs
    • System notifications
    • Scheduled reminders
  3. No configuration needed - works out of the box
